Программный комплекс для записи текста в картинку

Данная работы выполнена в рамках лабораторной работы по дисциплине «Информационная безопасность».

Порядок выполнения работы в программе для записи текста в картинку:

1)     выбрать вкладку “Записать в картинку”;

 

 

2)     нажать на кнопку “Обзор”;

3)     выбрать файл в формате PNG;

 

 

4)     после выбора будет активирована кнопка “Записать”, однако перед этим необходимо ввести исходный текст в поле ниже;

 

5)     нажать кнопку “Записать”;

6)     Далее необходимо будет выбрать место, куда сохранить новую картинку.

 

Порядок выполнения работы в программе для вытаскивания текста из картинки:

1)     выбрать вкладку “Вытащить текст из картинки”;

 

2)     нажать на кнопку “Обзор” ;

3)     выбрать файл в формате PNG;

4)     после выбора будет активирована кнопка “Вытащить” по нажатии на которую из картинки будет вытащен текст.
